# CrossEngine
<img src="CrossEngine/res/icon.ico"/>

## About
This is a *"game"* engine that is written purely in **C#**

## Features
- Supported platforms
    - Native
    - Web
- 2D/3D support
- Script system
    - Dynamic code loading
- Asset management
- Serialization system
- Service based architecture
- ECS based
    - Entities
    - Components
    - Component systems
- Abstract renderer API
- Built-in components
    - Cameras
    - Sprites
